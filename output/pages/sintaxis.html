Sintaxis

Comentarios

Los comentarios de una sola lína son definidos por el caracter #. Una vez se encuentra este caracter, el resto de la lína es omitida

variable = 12 # Esto es un comentario de una sola línea

Los comentarios de líneas múltiples se delimitan con las triples comillas '''

"""
Este es un comentario de múltiple
No es necesario definir ningún caracter extra
"""

Estructura de línea e identación

Cada sentencia es terminada con un caracter de salto de línea. Se puede separar una sentencia en varias líneas usando el caracter de continuación de línea \

valor = variable_1 + variable_2 * \
    (variable_3 + variable_4)

La identación adecuada es de suprema importancia para python. Bloques de código mal identados levantarán la exepción Identation Error.

if valor_booleano:
    sentencia
    sentencia  # Buena identación
else:
    sentencia
     sentencia # Mala identación

En el caso especial de los condicionales cortos, podemos tener una sintaxis como esta

if valor_booleano: sentencia
else: sentencia

Si se desea colocar mas de una sentencia en una línea, se usa como separador de sentencias el caracter ;

sentencia_1; sentencia_2; ...

Es posible mezclar identaciones entre la tecla tab y los espacios, siempre y cuando se mantenga el conteo de que cada tab va a tomar un de 8 espacios.

Palabras reservadas

En python es posible usar cualquier combinación iniciada con caracteres alfabéticos para definir identificadores.

Python es sensible a mayúsculas, por tanto variable es distinto de Variable y distinto de VARIABLE.

Palabras reservadas:
and elif global or
assert else if pass
break except import print
class exec in raise
continue finally is return
def for lambda try
del from not while

Es posible iniciar los identificadores con _ o con __ pero usualmente esta convención es usada para atributos privados en las clases y funciones o métodos especiales.

Literales

Existen 4 tipos de números
- Enteros: Definidos con un tamaño fijo en bytes
- Enteros grandes: Tamaño máximo definido por la cantidad de memoria disponible del equipo, se notan por una L al final
- Flotantes: Números en punto flotante
- Imaginarios: Números imaginarios, se notan por una J al final

Las cadenas de caracteres se pueden representar de dos maneras
- Caracteres de 8 bits: Usados usualmente para representar ASCII o latin1
- Caracteres de 16 bits: Usados para representar caracteres universalmente.

Operadores, delimitadores y símbolos especiales

Los operadores permitidos son:
+ (Suma) - (Resta) * (Multiplicación) ** (Potencia) / (División)
% (Módulo) >> (Shift derecho binario) << (Shift izquierdo binario) & (Conjunción binaria) | (Disyunción binaria)
^ (XOR binario) += (Suma y asignación) -= (Resta y asignación) *= (Multiplicación y asignación) **= (Potencia y asignación)
/= (División y asignación) %= (Módulo y asignación) <<= (Shift izquierdo y asignación) >>= (Shift derecho y asignación) &= (AND binario y asignación)
= (Asignación) ^= (XOR y asignación) ~ (Negación binaria) < (Menor que) > (Mayor que)
<= (Menor o igual que) >= (Mayor o igual que) == (Comparación) != (Diferencia) <> (Diferencia, Obsoleto en Python3)
