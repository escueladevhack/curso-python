El objeto None denota un objeto nulo, la representación de la nada, necesaria en los lenguajes de programación de alto nivel modernos. Python provee un único objeto nulo, que es escrito como None en un programa. Este objeto es retornado por funciones que no retornan explicitamente un valor. None es frecuentemente usado como el valor por defecto para argumentos no definidos explicitamente. Así, una función puede detectar si existe un valor para ese argumento o no.

None retorna False cuando se evalua como expresión booleana
